HyperText Markup Language

text = What you see
markup = what it looks like

Components of Markup

tags are composed of an opening tag, content and a closing tag.

< name > content < / name >

void tags have no conent and therfore don't need a closing tag.

HTML Attrubutes

anchors = < a > = make links

< tag attr = "value" > content < / tag >

Inline vs. Block

inline tags end a line and block tags create a box aorund the content.

Image tag

< img > = images

< img src = "url" alt = "text" >

the image tag has no content.
the alt text is displayed if the image doesn't load or for the blind.
